## User Stories

### Doctor
- As a doctor, I want to view my patients' medical history so that I can give better treatment.
- As a doctor, I want to update prescriptions so that patients receive accurate medication details.

### Patient
- As a patient, I want to book appointments online so that I can consult my doctor easily.
- As a patient, I want to view my medical records to track my health progress.

### Admin
- As an admin, I want to manage user accounts so that the system remains secure.
- As an admin, I want to generate reports to monitor system usage.
